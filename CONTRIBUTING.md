# Contributing

Thanks for your interest in contributing to DeliverXY.

## How to contribute
- Fork the repository
- Create a feature branch (`git checkout -b feature/my-feature`)
- Commit with clear messages
- Open a pull request with a short description

## Code style
- Backend: Java 17, Spring Boot conventions
- Frontend: Vue 3 / NativeScript-Vue 3, linted code
- Keep commits focused and readable

## Issues
Use GitHub Issues for bugs, improvements, or questions.
